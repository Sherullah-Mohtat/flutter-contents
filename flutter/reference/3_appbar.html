AppBar#

AppBar is a built-in widget that provides a material design app bar (also called a toolbar) at the top of your app‚Äôs screen. It‚Äôs commonly used inside a Scaffold widget and typically contains elements like a title, navigation icons, actions, and more.

Examples#

‚úÖ 1. leading:#

Displays a widget before the title (usually an icon or back button).

)

‚úÖ 2. automaticallyImplyLeading:#

)

‚úÖ 3. title:#

Main widget shown in the middle of the AppBar.

title: Text('My App'),
)

‚úÖ 4. actions:#

A list of widgets displayed at the end of the AppBar (right side).

)

‚úÖ 5. flexibleSpace:#

        ),
    ),
)

‚úÖ 6. bottom:#

Widget below the AppBar (commonly used for TabBar).

bottom: TabBar(
    tabs: [
        Tab(icon: Icon(Icons.home)),
        Tab(icon: Icon(Icons.settings)),
    ]),

‚úÖ 7. elevation:#

Z-axis shadow depth below the AppBar.

elevation: 4.0,
)

‚úÖ 8. shadowColor:#

Color of the AppBar‚Äôs shadow.

AppBar(
shadowColor: Colors.red,
)

‚úÖ 9. surfaceTintColor:#

Color to blend over the AppBar‚Äôs surface.

    surfaceTintColor: Colors.yellow,
)

‚úÖ 10. shape:#

Defines a shape for the AppBar.

    shape: RoundedRectangleBorder(
        borderRadius: BorderRadius.vertical(bottom: Radius.circular(30)),
    ),
)

‚úÖ 11. backgroundColor:#

The background color of the AppBar.

AppBar(
backgroundColor: Colors.purple,
)

‚úÖ 12. foregroundColor:#

The color for text and icons.

AppBar(
    foregroundColor: Colors.white,
)

‚úÖ 13. iconTheme:#

Theme for icons in the AppBar.

AppBar(
    iconTheme: IconThemeData(color: Colors.black),
)

‚úÖ 14. actionsIconTheme:#

Theme for action icons (right side).

AppBar(
    actionsIconTheme: IconThemeData(color: Colors.red),
)

‚úÖ 15. toolbarHeight:#

Height of the toolbar portion of the AppBar.

AppBar(
    toolbarHeight: 80.0,
)

‚úÖ 16. leadingWidth:#

Width of the leading widget space.

    leading: Icon(Icons.menu),
    leadingWidth: 100,
)

‚úÖ 17. toolbarOpacity:#

Opacity of the toolbar section.

    toolbarOpacity: 0.8,
)

‚úÖ 18. bottomOpacity:#

‚úÖ 19. toolbarTextStyle:#
